# pyeb
Python package for Entropy Balancing

## Installing 

*	Clone repository 

```
git clone https://github.com/bvegetabile/pyeb.git
```	

* From the directory with the repo

```
python -m pip install .
```	


## Example Usage

```python
from pyeb import sim_binary_data, entbal

dat = sim_binary_data(n_obs=5000, n_dim=10, seed=1234)
ta = dat.pop('TA')

eb = entbal()
eb.fit(dat, ta, estimand='ATE')
eb.print_balance()
```
